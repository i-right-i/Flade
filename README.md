# FLADE

**Fast LLM-Aided Design Environment**

FLADE is a planned Python coding agent built around a fast, keyboard-driven TUI for interactive software design and development.

Its core ideas include generated answer choices, compact task-specific context, configurable workflows, and mergeable sub-chats that let users explore design questions without cluttering the main conversation.

FLADE will reuse useful parts of the OpenHands Software Agent SDK while keeping its own session, context, workflow, permission, and mod systems.

> **Status:** Design phase. This repository is the future home of the project.
